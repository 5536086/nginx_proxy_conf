# nginx_proxy_conf
nginx任意反向代理配置(支持github,ip138,google)
这个是纯NGINX的任意反向代理
使用访问地址示例：
http://127.0.0.1/https://hostloc.com

我做这个的用处就是在某些不方便使用魔法的时候，下载东西，比如github里面的资源。
随手就是个github镜像站。



自定义路径 替换HostLocMJJ 为你自己的路径即可 有两处
